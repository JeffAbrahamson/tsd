Welcome to TSD

A time series is a set of (date, float) pairs. We say daily time series because it is a set of (date, float) pairs and not (time, float) pairs. So one value per day per series.

Typical time series might be daily readings of an electric meter, the reading on the bathroom scale in the morning, or the temperature at breakfast time.
